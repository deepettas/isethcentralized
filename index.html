Is eth centralized?

Posted on Thu 01 September 2022 in misc

Short answer: NOPE

Rationalle

The Ethereum network is a decentralized platform that runs smart contracts: applications that run exactly as programmed without any possibility of fraud or third-party interference. Ethereum is however facing scalability issues, with the number of transactions per second that can be processed limited by the network’s design. This has led to some members of the Ethereum community proposing to move to a new network that is based on a different design, called Ethereum 2.0. This debate is ongoing, with no clear consensus reached so far.
